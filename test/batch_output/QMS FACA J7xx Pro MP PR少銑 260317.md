## **Case：** QMS FACA J7xx Pro MP PR少銑 260317

## **Meta Data 元數據**

- 1. 檔案名：J7xx PR孔少銑

- 2. iPad機型：Pro

- 3. Build：MP

- 4. iPad機種：J71x

- 5. 製程：金加（CNC ）

- 6. 報告人：曾輝

- 7. 關鍵字：少銑


## **問題描述：**

BYD FATP 8/6回饋Shuke cell 灰PR孔槽少銑DQR碼：DQRHJX000150000VK1+6。


## **問題分析：**

1. 查詢不良品DQR碼DQRHJX000150000VK1+6過站記錄，Trace至CNC5線體B08機台，過站時間為2025/8/4 14:32，無重工記錄。
2. 排查B08機台當日首件紀錄，2025/8/4 12:15首件PR孔槽尺寸OK，無NG記錄，CMM檢測數據Pass。
3. 調閱刀具更換日誌，發現PR孔粗銑刀T4於8/4 13:50因主軸負荷異常更換，舊刀具存在刃口磨損跡象，懷疑更換前已產生少銑。
4. 查該機台8/4 13:50至14:30生產批次共42PCS，其中5PCS流入FATP，其餘在庫。
5. 少銑物理成因推演：T4刀具刃口磨損導致切削能力下降，未能完全銑除PR孔槽區域材料，且CNC6全檢未納入PR孔槽輪廓檢測，導致NG品漏出。


## **根本原因：**

1. 產生原因：PR孔粗銑刀T4刃口磨損導致切削能力下降，無法完全去除PR孔槽區域材料，造成漏銑（under-cut）。
2. 流出原因：CNC6全檢未納入PR孔槽輪廓檢測項目，AOI檢測範本未涵蓋該缺陷，導致少銑不良品未能於出站前攔截。


## **圍堵措施：**

1. 針對B08機台8/4 13:50至14:30生產之嫌疑批次42PCS進行全數管控：已出貨FATP之5PCS由FAE立即鎖碼攔截，隨線執行目視與CCD放大Sorting確認PR孔槽成型狀態；庫存37PCS於MCEG系統扣帳並鎖2D碼，安排專人100%目視全檢。
2. 在製品（WIP）部分，CNC5線體同機台後續批次於FQC系統攔截，確認無異常後方可續生產，防止異常擴大。
3. 针对CNC6全检漏失点，立即導入PR孔槽輪廓100% AOI全檢規則，同步更新檢測範本涵蓋少銑特徵，並由QE製作單點課程（OPL）對全檢人員實施教育訓練。
4. 所有經Sorting或重工OK之物料，出貨外箱標示「PR孔少銑全檢OK」並附檢人員簽核記錄，確保追溯性。


## **改善對策：**

1. 根因對策：生技優化PR孔粗銑刀T4刀具壽命管控，設定主軸負荷監控閾值，連續3次切削超標自動報警鎖機，並縮短T4刀具預防性更換週期20%，防止刃口磨損導致少銑。
2. 檢驗升級：將PR孔槽輪廓完整性納入CNC6 AOI 100%全檢項目，導入CCD高倍放大比對NG範本，系統自動攔截少銑異常品，並增設Go/No-go檢具做為人工複判標準。
3. 作業標準化：QE製作PR孔少銑單點課程(OPL)及不良樣板，包含刀具磨損前後對照圖，對IPQC、FQC及全檢人員實施教育訓練與盲測考核，確保辨識能力。
4. 品管卡控：OQC針對此異常加嚴抽驗，AQL抽樣標準由0.65加嚴至0.4，巡檢頻率提升為1次/2H，並追溯前3批次物料進行回溯確認。
