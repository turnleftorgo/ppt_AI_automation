## **Case：** QMS FACA J7xx Pro MP 電源孔過銑 260318

## **Meta Data 元數據**

- 1. 檔案名：J7xx 電源孔過銑

- 2. iPad機型：Pro

- 3. Build：MP

- 4. iPad機種：J71x

- 5. 製程：金加（CNC ）

- 6. 報告人：曾輝

- 7. 關鍵字：電源孔過銑


## **問題描述：**

CD FATP CQA抽檢發現1x Goku WF白HSG 電源孔過銑不良(DQRH7L0017200005NJ+A)。


## **問題分析：**

1. 查詢不良品DQRH7L0017200005NJ+A過站記錄，為B07線CNC6夾2025/04/15 14:30生產，機台編號E23；
2. 排查E23機台當日首件檢測記錄，4/15 14:00首件OK，無異常報警；
3. 調閱機台加工參數與刀具履歷，T12精修刀（W-DJ5122-Φ1.5R0.1）於4/15 12:00更換新刀，累計加工1,850孔，未達壽命上限（2,000孔）；
4. 分析過銑物理成因：精修刀具進刀量異常增大或主軸偏擺導致切削過深，可能為刀具微崩刃引發振動加劇；
5. 查E23機台4/15 14:30前後50片OMM檢測數據，僅該片電源孔尺寸超規，判定為偶發性刀具異常導致單件過銑；
6. 確認CNC6全檢項無電源孔尺寸自動檢測，依賴人工目檢，漏檢風險存在。


## **根本原因：**

1. 產生原因：T12精修刀具（W-DJ5122-Φ1.5R0.1）於加工過程中發生微崩刃，導致切削振動加劇、進刀量異常增大，造成電源孔局部過銑。
2. 流出原因：CNC6夾全檢站點未導入電源孔尺寸自動檢測（OMM/CCD），僅依賴人工目檢，無法有效識別微小過銑缺陷，導致不良品漏出。


## **圍堵措施：**

1. 針對B07線CNC6夾E23機台2025/04/15 14:30前後2小時生產之嫌疑物料共約90pcs進行管控：已出貨至FATP的50pcs由FAE立即鎖DQR碼並隨線Sorting，採CCD放大10X目視確認電源孔過銑狀態；廠內庫存約25pcs執行系統扣帳Hold並鎖2D碼，安排專人100%全檢；在製品15pcs於FQC站點攔截，未確認前禁止流入下一制程。
2. 所有Sorting動作依據不良形態使用CCD顯微鏡放大比對標準樣件，確認過銑深度與範圍，異常品隔離標示待FA分析。
3. 针对流出点漏检问题，立即在CNC6全检站導入電源孔區域100% CCD自動檢測，同步製作單點課程（OPL）教育訓練作業員辨識過銑特徵，OQC抽檢AQL由0.65加嚴至0.4並增加外觀複判機制。
4. 所有經全檢OK之物料，出貨外箱貼標識「電源孔過銑全檢OK」並註記檢驗日期與批次，確保追溯性。


## **改善對策：**

1. 治程防呆：針對T12精修刀（W-DJ5122-Φ1.5R0.1）增設加工振動監測系統，當振動值異常飆升＞3.0mm/s²時觸發即時報警並自動停機，防止微崩刃擴大過銑。
2. 檢驗升級：CNC6夾全檢站點導入電源孔區域100% CCD自動檢測，解析度5μm，比對NG範本自動判定過銑範圍與深度，系統攔截異常品並記錄影像軌跡。
3. 人員管理：QE立即製作電源孔過銑單點課程（OPL）與不良樣板，包含微崩刃導致過銑之實物圖示，對生產技工、IPQC、FQC實施教育訓練與盲測考核，通過率需達100%。
4. 品管卡控：OQC針對此不良項目啟動加嚴檢驗，AQL抽樣標準由0.65加嚴至0.4，並增設外觀複判機制（Double Check），每批隨機抽取10%由資深FQC複核確認。
