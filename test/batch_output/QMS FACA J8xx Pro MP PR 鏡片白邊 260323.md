## **Case：** QMS FACA J8xx Pro MP PR 鏡片白邊 260323

## **Meta Data 元數據**

- 1. 檔案名：J8xx  PR 鏡片白邊

- 2. iPad機型：Pro

- 3. Build：MP

- 4. iPad機種：J8xx

- 5. 製程：組裝（Assy）

- 6. 報告人：許榮

- 7. 關鍵字：PR 鏡片白邊


## **問題描述：**

BYD客戶端檢出7x PR鏡片白邊 配圖不良照片及客戶端反饋資訊。


## **問題分析：**

1.查詢不良物料SN過站記錄，Trace至Beita廠7/7、7/22及8/9三批，均為A1線體一次過站，無Rework記錄。
2.確認PR鏡片裝配工站為自動點膠機台，機台編號ASM-ADH-301~305，無機台集中性。
3.調閱點膠機參數報表，7/7與7/22批次膠量分布於1.9~2.3mg（中下限），8/9批次為2.0~2.6mg（中上限），標準範圍1.7~2.7mg，高膠量段對應溢膠風險。
4.比對CCD拍照影像，點膠路徑OK，無偏移或斷膠，但溢膠發生於熱熔後鏡片與HSG間隙，膠體爬升至外觀面。
5.排查裝配段差，96%數據≤0.03mm，排除組裝傾斜導致間隙過大。
6.判定白邊主因為熱熔膠過量填充後溢出外觀面，且FQC目檢標準未涵蓋間隙溢膠，導致漏檢流出。


## **根本原因：**

1. 產生原因：點膠制程膠量偏高（2.0~2.6mg）導致熱熔後膠體過量填充，爬升至PR鏡片與HSG間隙外觀面，造成白邊溢膠不良。
2. 流出原因：FQC外觀檢驗標準未納入間隙溢膠項目，目檢無法識別該缺陷，導致不良品漏檢流出。


## **圍堵措施：**

1. 排查Beita廠A1線體嫌疑批次：7/7（4pcs）、7/22（4pcs）、8/9（6pcs）共14pcs，已出貨至BYD端由FAE協同鎖機Sorting，採用CCD放大30X目視確認間隙溢膠狀況，全數檢驗OK後方可上線。
2. 廠內庫存部分同步扣帳Hold，共計約15K SN碼鎖2D碼系統攔截，安排專人於FQC站別進行CCD放大檢驗，確認無膠體爬升至外觀面後標記「PR白邊全檢OK」放行。
3. 在製品（WIP）即刻於FQC/OQC站別導入系統攔截規則，所有A1線體後段過站物料須通過間隙溢膠專項檢驗，異常品立即隔離並標示待判定。
4. 防堵流出點：FQC立即導入PR鏡片間隙溢膠之100%全檢標準，QE製作單點課程（OPL）完成全線檢驗員教育訓練，OQC抽檢AQL由0.65加嚴至0.4，出貨外箱明確標註「PR鏡片白邊全檢OK」識別標籤。


## **改善對策：**

1. 根因對策：生技針對ASM-ADH-301~305點膠機優化膠量參數，目標值由2.3mg下調至2.0±0.2mg，並納入SPC監控，每2H量測5點軌跡確認穩定性。
2. 防呆升級：機台新增膠量超出2.4mg自動報警鎖機功能，防止異常批量發生，同時導入熱熔後間隙溢膠CCD AOI檢測模組，自動比對NG範本攔截溢膠品。
3. 檢驗標準化：QE更新FQC外觀檢驗SOP，明確定義PR鏡片與HSG間隙不得有膠體爬升至外觀面，並製作3組實物不良樣板（含輕/中/重度溢膠）供比對。
4. 人員訓練：依據OPL教材對A1線IPQC/FQC全員實施溢膠識別教育訓練，完成後進行盲測考核，合格率須達100%方可上崗。
5. 品管卡控：OQC針對此項不良啟動為期30天重點管控，AQL抽檢標準由0.65加嚴至0.4，巡檢頻率提升為1次/2H，出貨箱貼「PR白邊全檢OK」識別標籤溯源。
