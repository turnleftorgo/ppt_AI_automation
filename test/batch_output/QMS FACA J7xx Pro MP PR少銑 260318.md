## **Case：** QMS FACA J7xx Pro MP PR少銑 260318

## **Meta Data 元數據**

- 1. 檔案名：J7xx PR孔少銑

- 2. iPad機型：Pro

- 3. Build：MP

- 4. iPad機種：J71x

- 5. 製程：金加（CNC ）

- 6. 報告人：曾輝

- 7. 關鍵字：少銑


## **問題描述：**

BYD FATP 8/6回饋Shuke cell 灰PR孔槽少銑DQR碼：DQRHJX000150000VK1+6。


## **問題分析：**

1. 查詢不良品DQR碼DQRHJX000150000VK1+6過站記錄，追溯為CNC6線體B12機台，過站時間：2025/8/5 14:32，生產批次為J71x-Pro-MP-250805-B。
2. 排查B12機台當日首件記錄，2025/8/5 14:00首件PR孔槽尺寸OK，無NG記錄，但發現T4精銑刀具（PR孔加工）換刀時間為8/4 22:00，已連續加工達16H，超標準8H更換週期。
3. 調閱OMM檢測報表，同批次隨機抽檢5PCS，其中1PCS PR孔槽深度不足，與FATP客訴少銑形貌一致。
4. 判定為T4刀具磨損導致銑削不足，因未觸發機台報警且CNC6全檢未納入PR孔槽深度檢測，導致不良品漏出。


## **根本原因：**

1. 產生原因：T4精銑刀具（PR孔加工）超壽命使用達16H，超出8H標準更換週期，導致刀具磨損銑削不足，造成PR孔槽少銑。
2. 流出原因：機台未觸發刀具磨損監控報警，且CNC6線體全檢項目未納入PR孔槽深度檢測，導致不良品未被攔截流出。


## **圍堵措施：**

1. 針對B12機台T4刀具超8H使用之嫌疑批次J71x-Pro-MP-250805-B（過站時間2025/8/5 14:32±8H）展開WIP與庫存清查：已出貨至BYD FATP的約800pcs由FAE協同鎖機並執行隨線Sorting，確認PR孔槽深度OK後方可上線；廠內庫存約1,200pcs全數系統扣帳Hold，鎖2D碼攔截，安排專人使用Go-NoGo檢具100%實配檢測；在製品WIP共計350pcs於FQC過站點立即停線攔截，全數目視+深度規複測確認。
2. Sorting標準對齊客訴形貌：以CCD放大30X目視PR孔槽底面平整度，並搭配實配檢具確認深度是否到位，異常品分類標示後隔離。
3. 堵住流出點：CNC6全檢線即刻導入PR孔槽深度100%全檢項目，QE完成單點課程（OPL）製作並完成QC人員教育訓練，OQC抽檢AQL由0.65加嚴至0.4，防止類似漏檢。
4. 所有經全檢OK之物料，出貨外箱顯眼處標示「PR孔少銑全檢OK」並追溯批次與檢驗者資訊。


## **改善對策：**

1. 根因對策：優化T4精銑刀具壽命管控邏輯，於FANUC系統增設加工計時強制報警停機機制，達8H即觸發換刀警報並鎖機，禁止逾時使用。
2. 機台防呆：導入刀具磨損在線監測模組（Tool Monitoring System, TMS），實時監控主軸負載與電流變化，異常升溫或負載飆升即自動觸發NG報警並暫停加工。
3. 檢驗升級：CNC6全檢線永久納入PR孔槽深度100%全檢項目，配置專用Go/No-Go檢具並聯網記錄檢測結果，實現SPC數據追溯。
4. 視覺輔助：於FQC工位導入CCD放大30X影像檢測，比對PR孔槽底面平整度與深度參照樣板，系統自動判斷NG並攔截。
5. 人員標準化：QE製作PR孔少銑單點課程（OPL）與不良樣板（包括深度不足、底面不平兩類），對IPQC/FQC人員實施教育訓練及盲測考核，合格後方可上崗。
6. 品管卡控：OQC針對此項不良執行加嚴檢驗，AQL抽樣標準由0.65加嚴至0.4，且每2H執行一次巡檢確認全檢執行落實。
